SceneJS > Examples > Message System Demo 1

This example requires a browser that supports the HTML5 <canvas> feature.

Sending "create" and "update" messages to SceneJS

In this example we're using the SceneJS message system to create and update nodes in the scene graph.

1. Add new subgraph - a Teapot wrapped in a Rotate, and add it to the scene graph.

2. Update Rotate.

3. Add another subgraph - a Teapot wrapped in a Translate, and add it to the scene graph.

4. Remove first subgraph.

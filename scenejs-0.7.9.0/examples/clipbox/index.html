Message API Plugin - "clip.clipbox"

The SceneJS API has an extensible messaging system which allows us to drive the JSON Scene Graph API via commands.

We can add more commands to the message API by installing our own command handlers.

This example demonstrates the "clip.clipbox" plugin, which defines a "clip.clipbox.create" command that creates a clip box around a target node within our scene graph and a "clip.clipbox.update" command which updates the extents of the clipbox.

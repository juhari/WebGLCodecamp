SceneJS Message API - Triggered Messages

The Message API from V0.7.9 onwards allows us to specify messages to be automatically sent whenever given trigger conditions occur. A trigger may be the elapse of a given time interval, or the occurrence of some observed event within SceneJS. For each trigger condition, we can specify whether the message is sent once for the first occurrence of the trigger, or every time the trigger occurs.
